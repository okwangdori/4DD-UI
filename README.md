# 홈페이지 URL

- https://fordd-dev.netlify.app

# 사용법

1.  yarn start - 실행
2.  'http://localhost:5173/' 로 접속
3.  스마트폰에서 접속시 (같은 네트워크망에서 가능) ipconfig로 IPv4 주소 확인 후 브라우저에서 'http://IPv4주소:5173/' 로 웹 접속
